What is Social Media Analytics?

(Gil Rael, 2-20-17)

Social Media Analytics is a Python Django web application developed by Gil Rael.

The application provides analytics on the following Social Media sites.

Facebook, Twitter, LinkedIN, YouTube, and Tumblr.

What other Python packages were used in the creation of Social Media Analytics?

(Gil Rael, 2-20-17)

Matplotlib - Python 2D plotting library, Numpy - Python scientific computing package.

Scipy - Python library for scientfic and technical computing, Tweepy - Twitter wrapper API

JSON - Python JSON package
